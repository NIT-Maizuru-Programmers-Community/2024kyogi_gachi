<h2>branch命名規則</h2>
<br>以下のような構成とする。</br>
<br><b>[第一]/[第二]/[第三]</b></br>

## 第一
<br>新機能"feat", デバッグ"debug"</br>
## 第二
<br>実験用"test", それ以外は省略可</br>

## 第三
<br>"プログラム名"</br>

## ブランチ名の例
<br>新機能で実験用の場合、"feat/test/hogehoge_abc"</br>
<br>デバッグの場合、"debug/error_xyz"</br>
